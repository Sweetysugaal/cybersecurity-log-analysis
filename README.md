# 🔐 Cybersecurity Log Analysis Project

## 📌 Overview
This project simulates a real-world cybersecurity task by analyzing system logs to detect suspicious activities such as failed login attempts and potential brute-force attacks.

It demonstrates practical skills in security monitoring, threat detection, and incident response.

---

## ⚙️ Tools & Technologies
- Python
- Log Analysis
- Git & GitHub

---

## 🚨 Key Features
- Detects multiple failed login attempts
- Identifies suspicious IP addresses
- Flags potential brute-force attacks
- Generates a structured incident report

---

## ▶️ How It Works
1. The script reads system log files  
2. It filters failed login attempts  
3. Counts attempts per IP address  
4. Flags suspicious activity based on thresholds
